# Ferrum 鐵鏈 · 國家級區塊鏈白皮書(網站)v1.2 雙語

以 Rust 打造的主權級區塊鏈白皮書。v1.2 新增**中／英即時切換**,
並新增一整章詳述**聯邦代幣治理的運作與鏈上實作**。

## 如何開啟
用瀏覽器開啟 `index.html`。單一自包含檔案;右上角 [中｜EN] 可即時切換語言。
字型線上載入,離線時自動退回系統字型,版面不受影響。

## 語言切換機制
中英內容成對存在,以 body class(lang-zh / lang-en)+ CSS 切換顯示;
連 SVG 圖內文字與程式碼註解皆雙語。預設繁體中文。
